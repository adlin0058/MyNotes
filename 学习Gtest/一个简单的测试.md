# Gtest测试

* 创建一个 **test.cc** 文件

 ```c++
 #include<gtest/gtest.h>
 //待测试代码
int add(int a,int b){
    return a+b;
}
//测试用例
TEST(testCase,test0){
    //如果符合预期就测试成功
    EXPECT_EQ(5,add(2,3));
    // 5为期望结果  add(2,3)为测试结果
    // 如果相等即为测试成功
}

int main(int argc,char **argv){
  testing::InitGoogleTest(&argc,argv);
  return RUN_ALL_TESTS();
}
 ```
 * 编译
```sh
g++ test.cpp -lgtest -lpthread
``` 
* 运行测试
```sh
./a.out
```
