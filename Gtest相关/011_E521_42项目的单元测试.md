### 单元测试
**测试函数**
```cpp
static uint8 e52142_port_rfc_is_ready(void){};
```

**函数用途**
检测两个e52142的RFC信号是否就绪

**函数输入**
1. 条件一 引脚状态
   类型 int 
   理想输入 1
2. 条件二 是否请求获取引脚状态
   类型 bool 
   理想输入 false
3. 条件三 检测两个RFC信号是否就绪
   类型 int
   理想输入 1
4. 条件四 RFC引脚状态的值
   类型 int
   理想输入 0x3

</font>

*条件输入*
```cpp

cd1 = false;
cd2 = 1;
cd3 = 0x3;
```
*预期结果*
```cpp
类型 int
输出 1
```
